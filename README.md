# bwm
iwctl wrapper for connecting to wireless networks

# Dependencies
glfw, imgui (included in this repo, no need to explicitly install)

# Installation

```
$ git clone --recursive https://github.com/Bananymous/bwm

$ cd bwm

$ premake5 gmake2

$ make config=release

# cp bin/Release/bwm /usr/local/bin/bwm
```
